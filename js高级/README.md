## 使用数组实现一个队列
    队列:是一种支持先进先出(FIFO)的容器,即先被插入的数据,先被取出!
    包含以下基本方法:
        enqueue(element): 加入队列
        dequeue(): 删除队列的第一个
        peek(): 查看队列的第一个
        size(): 队列的大小
        isEmpty(): 判断队列是否是空的
## 使用数组实现一个栈
    栈:是一种支持后进先出(LIFO)的容器,即后被插入的数据,先被取出!
    包含以下基本方法:
        push(element): 压栈
        pop(): 出栈
        peek(): 查看栈中的第一个
        size(): 栈的大小
        isEmpty(): 判断栈是否是空的
